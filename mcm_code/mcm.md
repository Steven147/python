## 数学建模学习交流

1. 评价类问题：评价目标、可选方案、**评价指标**
   1. 层次分析法：
      1. 权重表格：根据两两比较结果确定权重
      2. 判断矩阵：正负反矩阵，需要满足关系，成为一致矩阵（越不一致，最大特征值与n的差距越大）
      3. 一致性指标

          | 标度 | 含义          |
          | ---- | ------------- |
          | 1    | 同样重要/满意 |
          | 3    | 稍微重要/满意 |
          | 5    | 明显重要/满意 |
          | 7    | 强烈重要/满意 |
          | 9    | 极端重要/满意 |

      4. 对权重进行归一化（每一列得到一组权重）
         1. 算数平均
         2. 集合平均
         3. 特征值法：对特征向量进行归一化
